คำแนะนำและข้อควรปฏิบัติในการบันทึกไฟล์และออนไลน์

1. ตั้งชื่อไฟล์ให้ตรงกับสารบรรณ

เพื่อให้หน้าเว็บสารบรรณ (`index.html`) ตัวนี้กดลิงก์ได้อย่างถูกต้องและไม่มีปัญหาหน้า 404 Error ให้เซฟชื่อไฟล์ข้อสอบทั้ง 9 วิชาให้ถูกต้องตามรายชื่อดังนี้:

• พ.ร.บ. ป้องกันและบรรเทาสาธารณภัย พ.ศ. 2550.html
• ความรู้กรมปภ.html
• พ.ร.บ. วิธีการงบประมาณ พ.ศ. 2561.html
• ระเบียบ มท. ว่าด้วยกิจการ อปพร. พ.ศ. 2553.html
• พ.ร.บ. ปรับปรุงกระทรวง ทบวง กรม พ.ศ. 2545.html
• ระเบียบการป้องกันอุบัติภัยแห่งชาติ พ.ศ. 2564.html
• ระเบียบสำนักนายกรัฐมนตรี ป้องกันและลดอุบัติเหตุทางถนน พ.ศ. 2554.html
• ระเบียบการบริหารระบบการเตือนภัยพิบัติแห่งชาติ พ.ศ. 2552.html
• ระเบียบกระทรวงการคลัง ว่าด้วยเงินทดรองราชการ พ.ศ. 2568.html

2. การเปิดแชร์ออนไลน์ผ่าน GitHub Pages

หากท่านได้เอาชุดโปรแกรมข้อสอบนี้ขึ้น GitHub Repository และต้องการทำหน้าเว็บข้อสอบนี้ส่งต่อให้เพื่อนๆ เล่นผ่านมือถือหรือแท็บเล็ตได้สะดวก สามารถเปิดใช้งานได้ดังนี้:

  • ตรวจสอบให้แน่ใจว่าได้เปลี่ยนชื่อไฟล์สารบรรณหลักนี้เป็น `index.html` (ตัวเล็กทั้งหมด)
  • เข้าไปที่ระบบของท่านบน GitHub กดเลือกแท็บ Settings (เมนูด้านบนสุดรูปฟันเฟือง)
  • มองหาหัวข้อย่อยด้านซ้ายที่ชื่อว่า Pages ในหมวดหมู่ Code and automation
  • ที่ส่วนของ **Branch** ให้เปลี่ยนจาก `None` มาเลือกที่สาขาหลัก เช่น `main` หรือ `master` จากนั้นเลือก `/ (root)` แล้วกดบันทึก **Save**
  • รอประมาณ 1 - 2 นาที และเปิดดูลิงก์สีเขียวที่ระบบให้มาเพื่อใช้งานสอบออนไลน์ทันที!
